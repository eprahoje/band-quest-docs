# Feature 0007 - Band Members and Roster (Refinement 05)

## Decisions
- Cada um dos 15 personagens ganhou uma **história curta** (≈3 frases) para
  aproximar o jogador — o membro deixa de ser só "peça de build".
- **Tom narrativo: realista + bem-humorado**, com o equilíbrio variando por
  personagem para criar nuance e diferenciá-los entre si.
- As histórias são atemporais (não datam uma época) e coerentes com a função,
  o perfil de atributos e a identidade brasileira de cada um.
- Registradas em [`planning/design.md`](../planning/design.md) como identidade
  canônica; também servem de referência de **expressão** para a arte dos ícones.

## Questions
- Sem perguntas bloqueantes. Histórias seguem ajustáveis junto com nomes/perfis.
