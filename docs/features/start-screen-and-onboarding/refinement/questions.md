# Start Screen and Onboarding (Perguntas)

## Perguntas em aberto
- **Criação da Banda:** Os gêneros musicais disponíveis na criação dependem rigidamente da Era, ou o jogador é livre para escolher um gênero que não combina historicamente com a Era?
- **Criação da Banda:** O Traço de Origem (Perk inicial) impacta os atributos numéricos (Status base), a economia, ou a dificuldade do primeiro show?

## Perguntas resolvidas
- **A tela inicial terá animação ou será estática no MVP?** Terá uma animação simples no menu.
- **O onboarding será obrigatório ou opcional?** Será opcional. O jogador pode escolher se fará o tutorial ou não, e poderá tirar dúvidas das mecânicas durante o jogo a qualquer momento.

## Uso
- Registre aqui dúvidas bloqueantes antes de criar uma nova iteração.
