# Feature 0005 - Start Screen and Onboarding (Refinement 05)

## Foco da Iteração
Reconciliar o fluxo de nova partida com a redefinição da 0001 (sem seleção de
era) e a criação da 0010 (modos de sessão).

## Decisions
- **Não há tela de seleção de era** — o split "Era Selection and New Game Flow"
  foi descontinuado na 0001 (iteration-02).
- O fluxo de nova partida ganha uma etapa de **seleção de modo/duração de sessão**:
  temporizado (10/20/30 anos, com vitória/derrota) ou **modo livre** (duração
  personalizada). As *regras* dos modos vivem na **0010**; a *tela/fluxo* é desta
  feature.
- Nova fatia sugerida: **0005d — Session Mode Selection**, após 0005a–c.
- O desbloqueio de gêneros "por relevância histórica temporal" (iteration-03)
  passa a se apoiar na linha do tempo da **0008**.

## Questions
