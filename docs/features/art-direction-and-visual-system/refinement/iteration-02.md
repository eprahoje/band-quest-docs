# Feature 0006 - Art Direction and Visual System (Refinement 02)

## Decisions
- Os avatares dos membros da banda serão ilustrações vetoriais simples, uma
  pose fixa por personagem, sem animação.
- Essa escolha mantém o custo de arte baixo (uma única ilustração por
  personagem) enquanto oferece uma identidade visual mais forte que iniciais
  coloridas puras.
- A ilustração é estática e não requer variações por estado de humor/fadiga
  — esses estados continuam sendo representados por atributos numéricos.
