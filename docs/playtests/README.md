# Playtests / Feedback

Registro de sessões de playtest e do feedback do usuário sobre o
`band-quest-game`. Cada documento preserva o input original e mapeia cada ponto
para a feature dona (ou candidata), classificando como bug, balance, UX, nova
feature ou revisão de mecânica.

Esses pontos alimentam o `docs/features/roadmap.md` e os logs das features
correspondentes.
